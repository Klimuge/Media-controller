# MediaController

Компактный портативный медиаконтроллер на ESP32-S3 с механическими кнопками и Bluetooth.

## Что это
Устройство подключается к телефону или компьютеру по Bluetooth и управляет медиа:
следующий/предыдущий трек, пауза, громкость. Также поддерживает подключение по USB.
Заряжается через USB-C.

## Железо
- ESP32-S3-WROOM-1-N16R8
- Cherry MX Red — механические кнопки
- EC11 — энкодер для громкости
- LCD 2004 + I2C — дисплей
- LiPo 3.7В 3000мАч аккумулятор
- TP4056 — зарядка
- AP2112K-3.3 — LDO, питание логики 3.3V
- MTS-123 — тумблер переключения режимов

## Что сделано
- Схема и PCB в KiCad
- BOM с футпринтами
- 3д модели корпуса
- План кода

## Программная часть (не реализована)

Система состоит из трёх компонентов.

### Прошивка ESP32

NimBLE-Arduino (h2zero) — устройство представляется системе как BLE HID-клавиатура
и отправляет медиакейкоды. ESP32-S3 работает как составное USB-устройство одновременно:
HID (медиаклавиатура) + CDC (виртуальный COM-порт для агента).
ArduinoJson парсит входящие JSON-сообщения от агента и обновляет LCD.
Тумблер MTS-123 переключает режимы, состояние читается через GPIO38/GPIO39 с INPUT_PULLDOWN.
Энкодер EC11 управляет громкостью через прерывания.

### Агент

Фоновое приложение в системном трее. Читает метаданные текущего трека, системное время и т. д,
отправляет их на устройство по USB Serial (JSON). Если кабель не подключён то по Bluetooth. Также дожен запускать программу
если пользователь настроит запуск программы на кастомную (четвётую) кнопку

### PC-конфигуратор

GUI-приложение (PySide6). Пользователь настраивает устройство через интерфейс,
конфигуратор рендерит Jinja2-шаблон прошивки с этими настройками и автоматически
прошивает устройство через arduino-cli / esptool. Фактически — настройка и перепрошивка
без написания кода.
